What's New: Combining the Generic Scattering Calculator and Fitting for SAXS

Calculate excluded volume and hydration layers of structure files!

This adds support for specialized SAXS fitting that adds voids and waters to an atomic structure file (e.g. PDB), using a plugin model generated by the generic scattering calculator.

The calculator currently only supports X-ray scattering length densities. Neutron SLD support is planned for a future release.
