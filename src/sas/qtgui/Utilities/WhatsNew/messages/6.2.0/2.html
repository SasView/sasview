What's New: Combining the Generic Scattering Calculator and Fitting for SAXS

Calculate excluded volume and hydration layers of structure files!

Support for specialized SAXS fitting that adds voids and hydration layers to an atomic structure file (e.g. PDB), all using a plugin model generated by the generic scattering calculator.

The plugin model currently only supports X-ray scattering length densities. Neutron SLD support is planned for a future release.
